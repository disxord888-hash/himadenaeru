# Video Rank Analyzer

YouTubeのURLを貼り付けるだけで、再生数と経過日数から「人気度ランク」と「時期ランク」を自動判定するツールです。

## 判定基準

### 人気度ランク (Ω〜G)
1日あたりの再生数（再生回数 ÷ 経過日数）に基づいて判定されます。
- **Ω**: 1,000,000以上
- **UX**: 750,000以上
- **EX**: 500,000以上
- **ZZ**: 250,000以上
- **Z**: 100,000以上
- **S**: 50,000以上
- **A**: 10,000以上
- **B**: 5,000以上
- **C**: 1,000以上
- **D**: 500以上
- **E**: 250以上
- **F**: 100以上
- **G**: 100未満

### 時期ランク (N〜L3)
動画公開からの経過日数に基づいて判定されます。
- **N**: 0〜30日 (新作)
- **F**: 31〜90日 (フレッシュ)
- **R**: 91〜365日 (最近)
- **M1**: 1〜3年 (中期 前期)
- **M2**: 3〜5年 (中期 後期)
- **L1**: 5〜10年 (古典 初期)
- **L2**: 10〜15年 (古典 中期)
- **L3**: 15年以上 (歴史的古典)

## 使い方
1. `index.html` をブラウザで開きます。
2. YouTubeの動画URLを入力ボックスに貼り付けます。
3. 「分析する」ボタンを押します。
4. 自動取得が失敗した場合は、手動で再生数と公開日を入力することで判定可能です。
